Why this website exists

Browsing the modern day web kinda sucks. There's heaps of misinformation, ai generated "content" and piss-poor web design. I thought maybe I should try and provide an example of a modern, but well designed and easy to use website. I don't think we should have to return to the websites of the 1990s (see: spacejam's website) to achieve this. Using a bit of css. we can make websites that are tactile and intractable, without being confusing or overwhelming.
